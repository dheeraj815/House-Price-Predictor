# 🏠 House Price Prediction using Machine Learning

A machine learning–based web application that predicts house prices based on
multiple property features such as location, area, number of bedrooms,
bathrooms, and other relevant parameters.

## ✨ Features

- 🏘️ Predict house price based on location, area, bedrooms & bathrooms
- 📊 Visual charts for prediction results
- ⚙️ Feature engineering and scaling
- 🥇 Best ML model selected for accuracy
- ⚡ Real-time prediction results


## 🧠 Tech Stack

- Python
- scikit-learn
- Pandas
- NumPy
- Streamlit


## 📊 Models Used

- Linear Regression
- Random Forest Regressor
- XGBoost Regressor


## ▶️ Run Locally

git clone https://github.com/dheeraj815/House-Price-Predictor.git
cd House-Price-Predictor
pip install -r requirements.txt
streamlit run app.py
